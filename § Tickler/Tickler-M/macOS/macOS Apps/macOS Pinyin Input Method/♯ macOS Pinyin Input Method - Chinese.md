# macOS 拼音输入法

#macOS

## 双拼快捷键

* <kbd>⇧U</kbd>: 笔画输入
* <kbd>⇧␣</kbd>: 拆字，以及表情查询，笔画等菜单
* <kbd>⇧⌫</kbd>: 删除记忆字词
* <kbd>⌥</kbd>: 按住输入数字
* <kbd>⇧⌘[</kbd>: 按词选字（左）
* <kbd>⇧⌘]</kbd>: 按词选字（右）
* <kbd>⇧⌥B</kbd>: 符号和颜文字
* <kbd>⇧6</kbd>: 省略号以及最近使用符号
* <kbd>⇧{, ⇧}</kbd>: 直角引号
* <kbd>⇧⌥H</kbd>: 半角
* <kbd>⌃⇧␣</kbd>: 手写输入
* <kbd>⇧⌥L</kbd>: 编码查询

# Windows Shortcuts

#windows

⚡ System Tray

Press <kbd>Win+B</kbd> to activate tray, and use <kbd>Shift+F10</kbd> to show context menu.

⚡ Snip tool

<kbd>Win + Shift + S</kbd>

⚡ System Actions

<kbd>Win + x</kbd>

Combo examples:

* <kbd>Win +x, a</kbd>: Admin PowerShell
* <kbd>Win +x, u, s</kbd>: Sleep

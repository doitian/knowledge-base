# Aboboo

#windows #english

Aboboo is an English learning tool on Windows.

## Cheatsheet

* Ctrl + Left Click: Split
* Ctrl + Right Click at boundary: Merge
* Ctrl + Alt + Right Click: Delete

In Wave window:

* Left Click: Set Repeat Start
* Right Click: Set Repeat End

Playback Control:

* Esc: Next
* Backtick: Prev
* Tab: Pause/Play
* Ctrl + Enter: Replay
